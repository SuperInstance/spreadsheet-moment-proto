## Plugin Examples

### Custom Functions

#### Currency Converter

```typescript
=CURRENCY_CONVERT(100, "USD", "EUR")
```

**[View Plugin](https://github.com/spreadsheetmoment/plugins/currency-converter)**

#### Weather Data

```typescript
=WEATHER("New York", "temperature")
```

**[View Plugin](https://github.com/spreadsheetmoment/plugins/weather)**

#### Email Validation

```typescript
=IS_VALID_EMAIL("user@example.com")
```

**[View Plugin](https://github.com/spreadsheetmoment/plugins/email-validator)**

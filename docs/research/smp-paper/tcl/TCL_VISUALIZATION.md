### Type Inference Flow

```
Tile Definitions:
  parse:      String -> Transaction
  validate:   Transaction -> ValidatedTransaction
  transform:  ValidatedTransaction -> Decision

Inference:
  parse |> validate |> transform

Step 1: parse |> validate
  Input:  String
  Output: ValidatedTransaction (inferred)

Step 2: (parse |> validate) |> transform
  Input:  String
  Output: Decision (inferred)

Result: Flow type is String -> Decision
```

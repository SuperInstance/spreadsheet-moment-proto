### Competitive Moats

**1. Network Effects (Strong)**
- Users create agents → share with teams
- Public library grows with usage
- More agents → more value → more users
- Defensible against platform copycats

**2. Learning Data (Medium-Strong)**
- Agent performance data accumulates
- Proprietary training on "good agents"
- Improves with every user

**3. Switching Costs (Medium)**
- Users invest time in agent libraries
- Teams standardize on shared agents
- Migration path creates lock-in

**4. Multi-Platform (Medium-Strong)**
- Works across Excel, Sheets, Airtable
- Single platforms can't replicate
- Microsoft won't build for Google Sheets

**5. Technical Complexity (Medium-Strong)**
- Distillation requires ML expertise
- Multi-agent orchestration
- Open-source lacks productization

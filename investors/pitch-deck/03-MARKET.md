### Competitive Moats

**1. Network Effects**
- Users share agents → More users → More agents
- Public library grows with usage
- Defensible against platform copycats

**2. Learning Data**
- Agent performance data accumulates
- Proprietary training on "good agents"
- Improves with every user

**3. Switching Costs**
- Users invest time building agent libraries
- Teams standardize on shared workflows
- Migration creates lock-in

**4. Multi-Platform Strategy**
- Works across Excel, Sheets, Airtable
- Single platforms can't replicate
- Microsoft won't build for Google Sheets

**5. Technical Complexity**
- Distillation requires ML expertise
- Multi-agent orchestration
- Open-source alternatives lack productization

### Break-Even Analysis

**Fixed Costs:**
- R&D (base): $8M/year
- S&M (base): $4M/year
- G&A (base): $3M/year
- **Total Fixed: $15M/year ($1.25M/month)**

**Variable Costs:**
- COGS: 18% of revenue
- Variable OpEx: 20% of revenue
- **Total Variable: 38% of revenue**

**Break-Even Revenue:**
```
Fixed Costs / (1 - Variable %) = $15M / (1 - 0.38) = $24.2M/year
$24.2M / 12 = $2.02M/month
```

**Break-Even Users:**
$2.02M/month / $20 ARPU = 101K users

**Timeline:**
- Month 18: 100K users → near break-even
- Month 24: 1M users → profitable

### ORCID Profile Setup

**Step 1: Create Account**
1. Visit https://orcid.org
2. Click "Register now"
3. Choose "Individual" account
4. Enter personal information
5. Accept terms of use

**Step 2: Claim ORCID iD**
1. Record your 16-digit ORCID iD
2. Format: `0000-0000-0000-0000`
3. This is your permanent identifier
4. Use in all future publications

**Step 3: Add Publications**
1. Sign in to ORCID
2. Click "Works" → "+ Add works"
3. Add publications manually or via bulk import
4. Enter metadata for each paper
5. Link to arXiv and Zenodo once published

**Step 4: Link Accounts**
1. Link GitHub account
2. Link Zenodo account (will auto-sync after publication)
3. Link institutional email
4. Enable automatic updates

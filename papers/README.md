## Frequently Asked Questions

### Q: What's the difference between Zenodo and arXiv?

**A:** Both are academic repositories with different purposes:
- **Zenodo:** Assigns DOIs (permanent identifiers), better for final publication
- **arXiv:** Preprint server for rapid dissemination, better for visibility

**Recommendation:** Submit to both for maximum impact.

### Q: Can I submit papers individually or as a collection?

**A:** Both options available:
- **Zenodo:** Upload as collection (recommended) or individually
- **arXiv:** Must submit individually (one per submission)

**Recommendation:** Upload collection to Zenodo, submit individually to arXiv.

### Q: How long does the submission process take?

**A:** Timeline varies:
- **Zenodo:** 1-2 hours upload, 24-48 hours moderation, DOI assignment
- **arXiv:** 30 minutes per paper, 24-48 hours moderation per paper
- **Profiles:** 2-3 hours setup
- **Total:** 2-4 weeks for complete package submission

### Q: Do I need to convert Markdown to PDF?

**A:** Depends on repository:
- **Zenodo:** Accepts Markdown, no conversion needed
- **arXiv:** Requires LaTeX or PDF (convert Markdown → LaTeX → PDF recommended)

### Q: How do I track submission progress?

**A:** Use the tracking tools:
- **arXiv:** Update `papers/arxiv_package/ARXIV_SUBMISSIONS_INDEX.md`
- **Zenodo:** Record DOI in `papers/zenodo_package/zenodo_metadata.json`
- **Profiles:** Check citation counts and indexing status

### Q: Can I update papers after submission?

**A:** Different policies:
- **Zenodo:** Yes, create new version (preserves original DOI)
- **arXiv:** Yes, submit replacement (updates arXiv ID with version number)
- **Recommendation:** Finalize papers before submission

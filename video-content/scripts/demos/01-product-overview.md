## Production Notes

### Technical Requirements
- **Screen Resolution:** 1920x1080 (Full HD) minimum
- **Frame Rate:** 60fps for smooth animations
- **Color Scheme:** Dark theme with neon blue/green accents (matches brand)
- **Font:** Inter or Roboto for readability

### Screen Capture Settings
- **Zoom Level:** 125-150% for code readability
- **Mouse Cursor:** Enhanced visibility cursor
- **Highlighting:** Use subtle colored overlays for focus
- **Transitions:** Smooth dissolves (0.5s) between scenes

### Audio Requirements
- **Voiceover:** Professional voice talent, clear diction
- **Background Music:** Subtle electronic ambient (low volume)
- **Sound Effects:** Soft UI sounds for interactions
- **Mixing:** Voice at -6dB, music at -20dB

- [ ] Network visualization animation

# Round 15 White Paper Team - Lead Writer

## Mission: Paper Polish & Narrative Enhancement

### Primary Tasks:

1. **Narrative Arc Improvement**
   - Rewrite abstracts for maximum impact
   - Create compelling introductions that hook readers
   - Build progressive argument structures
   - Add real-world motivation and applications

2. **Clarity Enhancement**
   - Simplify complex mathematical exposition
   - Add intuitive explanations before formal definitions
   - Create running examples that span sections
   - Improve paragraph transitions and flow

3. **Academic Voice Refinement**
   - Ensure consistent academic tone
   - Eliminate colloquialisms and informal language
   - Strengthen argumentative structure
   - Add appropriate hedging and qualification

### Deliverables:
- `polished_abstracts_collection.md` - All abstracts rewritten
- `narrative_enhancement_guide.md` - Writing improvement guidelines
- `academic_voice_audit.md` - Tone consistency report
- `onboarding_whitepaper_lead_round15.md` - Knowledge transfer

### Style Focus:
- Active voice where appropriate
- Clear topic sentences
- Logical paragraph ordering
- Strong concluding statements

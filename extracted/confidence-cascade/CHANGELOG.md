# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-11

### Added
- Initial release of Confidence Cascade
- Three-zone confidence model (GREEN/YELLOW/RED)
- Sequential cascade type for chained decisions
- Parallel cascade type for weighted averaging
- Conditional cascade type for branched validation
- Comprehensive TypeScript support
- Event-driven architecture with callbacks
- Fraud detection example implementation

### Features
- Configurable confidence thresholds
- Automatic escalation triggers
- Detailed cascade results with debugging info
- Comprehensive test suite with >95% coverage
- Full API documentation

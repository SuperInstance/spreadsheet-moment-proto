## Primary Logo

### Logo Concept: "The Insight Grid"

**Visual Elements:**
1. **Grid Foundation** - Represents spreadsheet structure
2. **Moment Marker** - A highlighted cell showing the "moment" of insight
3. **Intelligence Glow** - Subtle radiance suggesting AI/tech
4. **Claw Sub-reference** - Optional subtle claw shape in negative space

**Technical Specifications:**
- **Format:** SVG (vector), PNG (raster)
- **Color:** Primary indigo with teal accent
- **Clear Space:** 1x height on all sides

## Logo Usage Guidelines

### Clear Space Requirements
Maintain clear space around the logo equal to the height of the "M" in "MOMENT" on all sides.

```
┌─┐
│ │← Clear space (M height)
│ │  ┌───┐
│ │  │Logo│
│ │  └───┘
│ │
└─┘
```

### Minimum Sizes

| Context | Minimum Width | Minimum Height |
|---------|--------------|----------------|
| **Digital (Screen)** | 120px | 40px |
| **Print (Business Card)** | 1.5" | 0.5" |
| **Print (Large Format)** | 2" | 0.67" |
| **App Icon** | 32px | 32px |
| **Favicon** | 16px | 16px |

### Do's and Don'ts

✅ **DO:**
- Use SVG format for digital applications
- Maintain aspect ratio (never stretch)
- Use appropriate variation for context
- Ensure sufficient contrast
- Maintain clear space
- Use official logo files only

❌ **DON'T:**
- Stretch or distort the logo
- Change logo colors
- Add effects or filters
- Rotate or skew the logo
- Place logo on busy backgrounds
- Create your own logo variations
- Use low-resolution files

---

## Logo Misuse Examples

### Incorrect Usage
1. **Stretching** - Never stretch disproportionate to dimensions
2. **Color Changes** - Never change brand colors to non-brand colors
3. **Background Conflicts** - Never place on low-contrast or busy backgrounds
4. **Effects** - Never add drop shadows, gradients, or filters
5. **Recreation** - Never recreate or modify logo elements
6. **Spacing** - Never crowd logo with other elements

### Corrective Actions
- Use original logo files from brand assets
- Choose appropriate logo variation for context
- Maintain proper clear space
- Test contrast and readability
- Use official color specifications

## Logo File Formats

### Vector Formats (Preferred)
- **SVG** - Web, scalable graphics
- **EPS** - Print, professional design
- **PDF** - Documents, print materials

### Raster Formats (When Necessary)
- **PNG** - Digital with transparency
- **JPEG** - Digital photography, print (no transparency needed)
- **WebP** - Modern web optimization

### Resolution Guidelines
- **Web:** 72-144 PPI (2x for retina)
- **Print:** 300 PPI minimum
- **Large Format:** 150-300 PPI depending on viewing distance

## Logo Contexts

### Website
- **Header:** Horizontal logo, left-aligned
- **Footer:** Stacked logo, left-aligned

### Applications
- **App Icon:** Icon only, rounded square

- **Business Cards:** Stacked logo, front
- **Letterhead:** Horizontal logo, top-left

### Social Media
- **Profile Picture:** Icon only, 400x400px minimum
- **Cover Photo:** Wordmark or horizontal logo

### Merchandise
- **T-Shirts:** Full color or outline, centered

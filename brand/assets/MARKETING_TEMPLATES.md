- **Title:** Display 2 (60px), Cal Sans
- **Subtitle:** Heading 4 (20px), Inter
- **Body:** Body (16px), Inter
- **Caption:** Caption (12px), Inter

### Color Scheme
- **Background:** White or very light gray
- **Text:** Dark gray (#111827)
- **Accent:** Primary indigo (#4F46E5)
- **Highlights:** Secondary teal (#0D9488)

### Usage Guidelines
- Use consistent layouts
- Maintain visual hierarchy
- Limit text per slide (3-5 bullets)
- Use high-quality images (minimum 150 DPI)
- Include logo on every slide
- Number slides for reference

## File Format Specifications

### Print Assets
- **Vector:** AI, EPS, SVG (for logos and illustrations)
- **Raster:** TIFF, PNG (300 DPI minimum)
- **Documents:** PDF, INDD
- **Color Mode:** CMYK

### Digital Assets
- **Images:** PNG (with transparency), JPEG (photos)
- **Graphics:** SVG (icons and logos), WebP (modern web)
- **Video:** MP4 (H.264), WebM
- **Documents:** PDF
- **Color Mode:** RGB

### Font Files
- **Web:** WOFF2, WOFF
- **Desktop:** OTF, TTF
- **Print:** OTF

## Custom Template Requests

### Process
1. Submit request to marketing team
2. Specify use case and requirements
3. Design team creates initial mockup
4. Review and revision cycle
5. Final approval and delivery

### Timeline
- **Simple templates:** 2-3 business days
- **Complex templates:** 1-2 weeks
- **Rush requests:** 24-48 hours (additional cost)

## Podcast Appearance Checklist

### Pre-Appearance Preparation

**Research:**
- [ ] Listen to 2-3 recent episodes
- [ ] Understand podcast format and style
- [ ] Identify host's interviewing style
- [ ] Note common topics and themes
- [ ] Understand audience demographics

**Content Preparation:**
- [ ] Prepare 3-5 key talking points
- [ ] Prepare stories and examples
- [ ] Prepare data and statistics
- [ ] Prepare customer stories
- [ ] Prepare "aha!" moments

**Technical Preparation:**
- [ ] Test microphone and camera
- [ ] Test internet connection
- [ ] Prepare backup recording setup
- [ ] Have headset with microphone
- [ ] Test recording software/platform

**Logistics:**
- [ ] Confirm date and time (timezone!)
- [ ] Confirm recording platform (Zoom, Riverside, etc.)
- [ ] Confirm episode length
- [ ] Confirm release timeline
- [ ] Add to calendar with reminders

### During Appearance

**Opening:**
- [ ] Thank host for having you
- [ ] Brief intro (15-30 seconds)
- [ ] Set expectations for discussion
- [ ] Establish rapport with host

**Content:**
- [ ] Stick to prepared talking points
- [ ] Tell stories, not just facts
- [ ] Use concrete examples
- [ ] Keep answers concise (2-3 minutes)
- [ ] Be authentic and conversational

**Closing:**
- [ ] Summarize key takeaways
- [ ] Share where to learn more (website, GitHub)
- [ ] Mention social media handles
- [ ] Thank host and audience
- [ ] Offer to follow up on additional topics

### Post-Appearance Follow-up

**Immediate (Same Day):**
- [ ] Send thank you email to host
- [ ] Share episode on social media
- [ ] Tag host and podcast in posts
- [ ] Provide additional resources if requested

**Week of Publication:**
- [ ] Share episode across all channels
- [ ] Encourage team to share
- [ ] Respond to comments and mentions
- [ ] Monitor downloads and engagement

**Post-Publication:**
- [ ] Send additional thank you to host
- [ ] Ask for metrics (downloads, reach)
- [ ] Offer to return for future episodes
- [ ] Maintain relationship with host

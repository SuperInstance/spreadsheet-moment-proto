# Spreadsheet Moment - Migration Guides

Welcome to the comprehensive migration guide for switching to Spreadsheet Moment from competing platforms. These guides are designed to make your transition smooth, efficient, and painless.

## Why Migrate to Spreadsheet Moment?

Spreadsheet Moment offers several advantages over traditional spreadsheet and database platforms:

- **AI-Native Integration**: Built-in cell agents that understand context and automate complex tasks
- **Real-Time Collaboration**: WebRTC-based real-time collaboration with sub-second sync
- **Self-Hosted Privacy**: Full control over your data with self-hosting options
- **Modern Architecture**: Built with React, WebAssembly, and Rust for performance
- **Extensible Platform**: Plugin system and API for custom workflows
- **Cost Effective**: No subscription fees or usage limits when self-hosted

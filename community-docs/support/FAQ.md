## Getting Started

### What is Spreadsheet Moment?

Spreadsheet Moment is a next-generation collaborative spreadsheet platform that combines traditional spreadsheet functionality with modern collaboration features, AI assistance, and a vibrant community ecosystem.

### Is Spreadsheet Moment free?

Yes! We offer a generous free tier that includes:
- Unlimited spreadsheets
- Basic collaboration features
- Community templates
- Core AI features

Paid plans are available for advanced features, more storage, and priority support.

### How do I get started?

1. **Create an account** at spreadsheetmoment.com
2. **Complete your profile** - Add a photo and bio
3. **Take the tour** - Learn the basics
4. **Join the community** - Discord, forum, etc.
5. **Start creating** - Make your first spreadsheet!

See our [Getting Started Checklist](../onboarding/CHECKLIST.md) for a detailed guide.

### What browsers are supported?

We support the latest versions of:
- Google Chrome (recommended)
- Mozilla Firefox
- Safari
- Microsoft Edge

For the best experience, keep your browser updated.

### Is there a mobile app?

We're working on native mobile apps! In the meantime, our web interface is fully responsive and works great on mobile browsers.

## Account & Profile

### How do I create an account?

Visit [spreadsheetmoment.com/signup](https://spreadsheetmoment.com/signup) and provide:
- Email address
- Password
- Display name

You'll receive a verification email to confirm your account.

### I forgot my password. What do I do?

Click "Forgot password" on the login page and enter your email address. You'll receive a link to reset your password.

### Can I change my display name?

Yes! Go to:
1. Settings (click your avatar)
2. Profile
3. Edit display name
4. Save changes

Your display name can be changed once every 30 days.

### How do I delete my account?

Go to:
1. Settings
2. Account
3. Delete Account
4. Follow the confirmation process

**Note**: This action is irreversible. You'll lose all your spreadsheets and data.

### Can I merge two accounts?

Currently, we don't support account merging. However, you can export data from one account and import it to another.

## Using the Platform

### How do I create a new spreadsheet?

Click the "New Spreadsheet" button on your dashboard or use the shortcut `Ctrl+N` (Windows) or `Cmd+N` (Mac).

### How do I share a spreadsheet?

1. Open the spreadsheet you want to share
2. Click "Share" button (top right)
3. Choose sharing method:
   - **Invite people** - Enter email addresses
   - **Get link** - Create shareable link
   - **Export** - Download as file
4. Set permissions (view, comment, edit)
5. Send or save the link

### Can I collaborate in real-time?

Yes! Multiple people can edit the same spreadsheet simultaneously. You'll see:
- **Cursors** - See where others are working
- **Updates** - Changes appear in real-time
- **Comments** - Chat and discuss inline
- **Presence** - See who's viewing

### How do I use formulas?

Just like traditional spreadsheets:
1. Click a cell
2. Type `=` to start a formula
3. Use functions like `SUM()`, `AVERAGE()`, etc.
4. Press Enter to calculate

See our [Formula Reference](https://spreadsheetmoment.com/docs/formulas) for all available functions.

### What AI features are available?

Free tier:
- **Smart autocomplete** - Formula suggestions
- **Error detection** - Find and fix errors
- **Basic insights** - Simple data analysis

Paid tier:
- **Advanced analytics** - Deep data insights
- **Natural language** - Ask questions in plain English
- **Predictive modeling** - Forecast and trends
- **Data cleaning** - Automatic data fixing

### How do I import data?

1. Open your spreadsheet or create new
2. Click "Import" (File menu or toolbar)
3. Choose file type:
   - CSV
   - Excel (.xlsx, .xls)
   - Google Sheets
   - Paste from clipboard
4. Follow the import wizard

### Can I export my data?

Yes! Export to:
- PDF
- Excel (.xlsx)
- CSV
- HTML
- JSON

Go to File > Export and choose your format.

## Contributing

### How can I contribute?

There are many ways to contribute:
- **Code** - Fix bugs, add features
- **Documentation** - Improve docs, write guides
- **Design** - UI/UX improvements
- **Community** - Help users, organize events
- **Translation** - Translate to other languages
- **Testing** - Test new features

See [Contributor Guide](../contributing/README.md) for details.

### Do I need to be a developer to contribute?

No! We welcome non-code contributions:
- Help users in the forum
- Improve documentation
- Report bugs
- Share templates
- Translate content
- Design improvements

See [First Contribution Guide](../contributing/FIRST_CONTRIBUTION.md) for beginner-friendly ways to start.

### What's the contribution process?

1. **Find an issue** or propose an idea
2. **Claim the issue** (comment that you're working on it)
3. **Create a branch** from main
4. **Make your changes**
5. **Test thoroughly**
6. **Submit a pull request**
7. **Address feedback**
8. **Celebrate when merged!** 🎉

See [PR Workflow](../contributing/PR_WORKFLOW.md) for detailed steps.

### How do I report a bug?

1. **Search existing issues** first
2. **Create a new issue** on GitHub
3. **Use the bug report template**
4. **Provide details**:
   - Steps to reproduce
   - Expected behavior
   - Actual behavior
   - Screenshots if relevant
   - Environment info

See [Issue Reporting](../contributing/ISSUES.md) for more details.

## Technical Support

### Something isn't working. What do I do?

Try these steps:
1. **Refresh the page** - Sometimes that's all it takes
2. **Check your browser** - Use a supported browser
3. **Clear cache** - Clear browser cache and cookies
4. **Check status** - Visit [status.spreadsheetmoment.com](https://status.spreadsheetmoment.com)
5. **Search the FAQ** - You're here now!
6. **Ask for help** - Forum or Discord

### My spreadsheet won't load. Help!

Try:
1. **Check your internet** connection
2. **Try a different browser**
3. **Clear your cache**
4. **Disable extensions** - Some extensions interfere
5. **Check the status page** for outages

Still not working? Contact support with:
- Your browser and version
- Operating system
- Error messages (if any)
- Steps to reproduce

### I found a security issue. How do I report it?

**Please don't publicly disclose security issues!**

Email: security@spreadsheetmoment.com

We'll respond within 24 hours and work with you to address the issue.

## Privacy & Security

### Is my data secure?

Yes! We use:
- **Encryption** - Data encrypted in transit and at rest
- **Access controls** - You control who sees your data
- **Backups** - Daily backups with disaster recovery
- **Compliance** - GDPR, CCPA compliant

See our [Security Whitepaper](https://spreadsheetmoment.com/security) for technical details.

### Who can see my spreadsheets?

Only people you explicitly share with. We never:
- Sell your data
- Access your spreadsheets without permission
- Use your data for training (without explicit consent)
- Share data with third parties

### Where is my data stored?

We use secure cloud infrastructure with data centers in:
- United States (us-east, us-west)
- Europe (eu-central)
- Asia Pacific (ap-southeast)

You can choose your data region in settings.

### Do you use my data to train AI?

Only if you opt in. By default:
- **Your data is not used** for AI training
- **You can opt in** to help improve AI features
- **Aggregate, anonymized data** may be used for analytics
- **You can opt out anytime** in settings

### Is Spreadsheet Moment GDPR compliant?

Yes! We comply with GDPR and other privacy regulations. You can:
- Export your data
- Delete your account
- See what data we have
- Object to processing
- See [Privacy Policy](https://spreadsheetmoment.com/privacy)

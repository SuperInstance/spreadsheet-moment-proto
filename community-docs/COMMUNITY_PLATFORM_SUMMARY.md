## Usage Guidelines

### For New Community Members
1. Start with `onboarding/README.md`
2. Complete the `onboarding/CHECKLIST.md`
3. Read `onboarding/GUIDELINES.md` and `onboarding/CODE_OF_CONDUCT.md`
4. Explore `onboarding/COMMUNICATION.md` to find your channels
5. Choose your contribution path from `contributing/README.md`

### For Contributors
1. Review `contributing/README.md` for overview
2. Follow `contributing/FIRST_CONTRIBUTION.md` for your first contribution
3. Study `contributing/PR_WORKFLOW.md` for pull requests
4. Check `docs/STANDARDS.md` if contributing documentation
5. Track your progress in `recognitions/TIERS.md` (when created)

### For Community Managers
1. Use `programs/README.md` to understand program structure
2. Implement `engagement/CHALLENGES.md` for weekly activities
3. Reference `support/FAQ.md` for common questions
4. Follow `onboarding/CODE_OF_CONDUCT.md` for enforcement
5. Customize templates as needed for your community

### For Program Coordinators
1. Review your specific program page under `programs/`
2. Adapt guidelines to your program's needs
3. Track metrics mentioned in program overviews
4. Coordinate with other programs using shared resources
5. Report impact using suggested metrics

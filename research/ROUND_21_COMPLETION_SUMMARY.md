## Known Limitations

### Minor Issues (Non-blocking)
1. **AVIF format support:** Limited to modern browsers (Safari fallback to WebP)
2. **Offline mode:** Full offline support requires additional service worker configuration
3. **Real-time collaboration:** Limited to 100 concurrent users per spreadsheet (can be increased)
4. **AI features:** Requires API keys for GPT-4 and other AI services
5. **Distributed computation:** Requires minimum 3 nodes for fault tolerance

### Future Enhancements (Out of Scope)
1. **Desktop apps:** Electron/Tauri packaging
2. **Plugin marketplace:** Community publishing workflow
3. **Advanced AI features:** Code completion, natural language queries
4. **Mobile app store deployment:** iOS App Store and Google Play
5. **Enterprise SSO:** Additional SAML/OIDC providers

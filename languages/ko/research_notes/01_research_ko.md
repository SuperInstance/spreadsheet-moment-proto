# 한국어 번역 연구 노트: SuperInstance: The Universal Cell Architecture

## 📚 한국어 학술 컨텍스트 분석

### 한국 수학 및 컴퓨터 과학 학술 전통
- **체계적 접근**: 한국 학술 논문은 체계적이고 논리적인 구조를 선호함
- **정밀한 용어**: 기술 용어에 대한 정확한 번역이 중요함
- **교육적 명확성**: 복잡한 개념을 단계적으로 설명하는 스타일
- **국제적 표준**: 수학 표기법은 국제 표준을 따르며 LaTeX 사용이 일반적

### 문화적 적응 필요 영역

#### 1. 사례 연구 (Case Studies)
- **원문**: 미국 주식 시장 예시 (stock price monitoring)
- **한국 적응**: KOSPI(코스피) 지수 또는 삼성전자 주가 모니터링 예시로 변경
- **이유**: 한국 독자에게 더 친숙한 예시 제공

#### 2. 기술 용어 번역
- **SuperInstance**: "슈퍼인스턴스" (외래어 표기) 또는 "초인스턴스" (한자어)
- **Rate-based change**: "율 기반 변화" 또는 "변화율 기반"
- **Confidence cascade**: "신뢰도 캐스케이드" 또는 "신뢰도 전파"
- **Origin-centric**: "원점 중심" 또는 "기준점 중심"

#### 3. 학술 스타일 적응
- **문장 구조**: 한국어는 주어-목적어-서술어 순서 (SOV)
- **수동태**: 영어보다 덜 사용됨
- **공식성**: 학술 논문은 존댓말(해요체) 사용

#### 4. 수학 표기법
- **LaTeX 보존**: 모든 수학 표기법은 원문 그대로 유지
- **변수 이름**: x, y, z, α, β, γ 등 그대로 유지
- **증명 구조**: 논리적 단계는 그대로 유지

## 🎯 번역 전략

### 1. 수학적 정확성 우선
- 모든 방정식, 증명, 수학적 구조는 원문 그대로 보존
- LaTeX 코드는 변경 없이 유지
- 변수 이름은 영어 그대로 사용

### 2. 문화적 적응
- 미국 중심 예시 → 한국 독자 친숙한 예시로 변경
- 한국 학술 논문 스타일 적용
- 한국어 자연스러운 문장 구조 사용

### 3. 용어 일관성
- 전체 논문에서 동일한 용어 사용
- 신조어는 각주로 영어 원문 표기
- 복잡한 개념은 설명 추가

### 4. 가독성
- 긴 영어 문장 → 짧은 한국어 문장으로 분리
- 복잡한 개념은 예시와 함께 설명
- 학술적이되 이해하기 쉽게 번역

# 🎓 Cross-Cultural Classroom Simulation Framework
## 25-Round Educational Dialogue System

### 🌍 Framework Overview
This framework simulates classroom-style dialogues with Socratic teaching methods across cultures, languages, and learning levels. It generates ML training data for educational AI systems.

### 🎯 Simulation Structure

#### Rounds 1-5: Teacher-Guided Cross-Cultural Dialogues
- Socratic teachers engaging with diverse students
- Cultural exchange and different learning approaches
- Progressive complexity across topics

#### Rounds 6-10: Teacher-to-Student Agent Conversations
- Direct teaching from teacher agents to student agents
- Personalized learning paths based on cultural background
- Adaptive teaching methods

#### Rounds 11-15: Peer Learning Groups
- Student agents teaching each other
- Cross-cultural collaboration
- Different age groups and professions

#### Rounds 16-20: Professional Development Groups
- Working professionals in various fields
- Industry-specific applications
- Advanced technical discussions

#### Rounds 21-25: Synthesis and Meta-Learning
- Analysis of teaching methods
- Cultural adaptation patterns
- ML training data generation

### 🎭 Character Roles

#### Teachers (Socratic Method)
1. **Dr. Chen** - Mathematics professor, systematic approach

### 🗣️ Dialogue Patterns

#### Socratic Questioning Techniques
1. **Clarification Questions** - "What do you mean by...?"
2. **Assumption Probing** - "What are we assuming here?"
3. **Evidence Requests** - "How do we know that's true?"
4. **Viewpoint Exploration** - "What alternative perspectives exist?"
5. **Implication Analysis** - "What are the consequences?"
6. **Meta-Cognitive Prompts** - "How did you arrive at that conclusion?"

#### Cultural Adaptation Strategies
1. **Metaphor Translation** - Convert concepts to cultural contexts
2. **Learning Style Matching** - Visual, auditory, kinesthetic, analytical
3. **Language Bridge-Building** - Multilingual explanations
4. **Historical Contextualization** - Connect to cultural knowledge
5. **Collaborative Learning** - Peer teaching across cultures
